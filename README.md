# Thiranex Task 4 - Django Blog Platform with Comments

**Developer:** Tejasri  
**Date:** June 2026  
**Tech Stack:** Django 5.0, Python 3.11, SQLite, HTML5, CSS3

## 📋 Project Overview
A full-stack blogging platform built for Thiranex Task 4. Users can view blog posts and add comments after authentication. Admin panel supports full CRUD operations for content management.

## ✅ Key Features Implemented

| Feature | Status | Details |
| --- | --- | --- |
| **Backend & Database** | ✅ Complete | Django Models with SQLite. Post + Comment models with ForeignKey relation |
| **Blog CRUD** | ✅ Complete | Create, Edit, Delete posts via Django Admin Panel |
| **Comment Section** | ✅ Complete | Authenticated users can add comments on posts |
| **User Authentication** | ✅ Complete | Django's built-in Admin authentication system |
| **Frontend** | ✅ Complete | Responsive HTML templates for Post List & Post Detail views |
